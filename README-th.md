## 🌟 ภาพรวม

โปรเจกต์นี้คือซอร์สโค้ดสำหรับ **Interactive Web Resume** ของผม ซึ่งสร้างขึ้นเพื่อก้าวข้ามขีดจำกัดของเรซูเม่รูปแบบ PDF แบบเดิมๆ โดยนำเสนอข้อมูลประวัติการทำงาน ทักษะ และโปรเจกต์ต่างๆ ในรูปแบบที่สะอาดตา ตอบสนองได้รวดเร็ว (Responsive) และน่าสนใจสำหรับผู้สรรหาบุคลากร (Recruiters)

ออกแบบด้วยสไตล์ที่ทันสมัย เน้นความอ่านง่าย และโครงสร้างข้อมูลที่ชัดเจน พร้อมการโหลดที่รวดเร็วเป็นพิเศษ

## 🚀 ฟีเจอร์หลัก

- **Mobile Responsive:** รองรับการใช้งานอย่างสมบูรณ์แบบทั้งบนคอมพิวเตอร์ แท็บเล็ต และสมาร์ทโฟน
- **Print-Friendly:** มีการใช้ CSS Media Queries (`@media print`) เพื่อให้แน่ใจว่าเมื่อพิมพ์ออกมาเป็นกระดาษหรือ PDF จะมีความสะอาดตาและเป็นระเบียบ โดยตัดส่วนที่ไม่จำเป็นออกอัตโนมัติ
- **Modern UI/UX:** ใช้ตัวอักษรที่สะอาดตา ลำดับความสำคัญของข้อมูลชัดเจน และเอฟเฟกต์ Hover ที่ช่วยให้นำสายตาผู้อ่านได้ดี
- **Quick Links:** เชื่อมโยงไปยัง Portfolio, GitHub และช่องทางการติดต่อได้อย่างรวดเร็ว
